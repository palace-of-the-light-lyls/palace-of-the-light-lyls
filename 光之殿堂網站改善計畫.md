# 光之殿堂網站改善計畫

更新日期：2026-06-16

## 目前階段

Refinement｜網站完善檢查與展示體驗強化

## 改善目標

1. 版權字樣正確化
   - 將圖片版權字樣依素材性質分類，避免所有圖片都使用「3D 示意圖」這類不精準說法。
   - 空間圖、材質／工藝圖、面板圖、動態示意、安藤作品參考圖需使用不同標示。
   - 清除 CSS 中早期殘留的舊版權字樣，避免後續樣式調整時錯誤文字再次出現。

2. 手機展示體驗強化
   - 商品系列 tab 在手機上需清楚提示可左右滑動。
   - 手機抽屜選單進入商品頁後，需正確切換商品、關閉選單並同步 active 狀態。
   - 業務拿手機展示時，應能快速切換六大商品，不需要解釋操作方式。

3. 網站穩定性與交接性
   - 修改以小範圍、低維護成本為原則。
   - 不改動網站主架構，不新增大型功能。
   - 每次調整後需更新 CSS 版本號，避免預覽或上線時讀到舊快取。

4. 配置圖與放大檢視強化
   - 手機點開商品區位圖或空間區位圖後，需清楚知道可以放大、拖曳與返回原頁。
   - 放大視窗需保留明顯關閉按鈕，避免業務展示時卡在圖片頁。
   - 關閉後需回到原商品頁，不影響目前選到的商品。

5. 手機長頁閱讀節奏
   - 商品頁在手機上不宜讓設計團隊內容連續垂直堆疊過長。
   - 多位設計／工藝團隊時，改用橫向卡片展示，降低單一路徑滑動長度。
   - 只有一張團隊卡時，不顯示「左右滑動」提示，避免誤導。

6. 手機商品規格與配置圖段落
   - 規格表需維持清楚可讀，不為縮短頁面而犧牲資訊理解。
   - 配置圖卡片在手機上不宜垂直堆疊過長，需改為橫向滑動展示。
   - 點擊配置圖後仍需正常開啟放大視窗。

7. 手機頁尾與回到上方動線
   - 長頁滑到底後，不應只剩說明文字，需提供清楚下一步。
   - 頁尾需提供回到商品系列、查看工程進度、回到首頁等快速導覽。
   - 回到上方按鈕需可見且不遮住頁尾操作按鈕。

## 已完成項目

- 版權字樣已改為依圖片性質分類：
  - 空間示意圖
  - 空間概念示意圖
  - 空間動態示意
  - 材質／工藝示意圖
  - 材質／工藝參考圖
  - 面板設計示意圖
  - 安藤忠雄作品參考圖
- 舊版「此為3D示意圖，商品以實際為準」已從畫面與 CSS 殘留規則中清除。
- 手機版商品 tab 已新增「左右滑動查看更多商品」提示。
- 手機版商品 tab 右側已保留漸層提示與滑動空間，降低使用者漏看後方商品的機率。
- 配置圖放大視窗已新增操作提示：「手機可雙指縮放、拖曳查看細節；點右上角關閉回到原頁。」
- 配置圖放大視窗已補強關閉機制：右上角關閉按鈕可返回原頁，背景關閉也加上程式保險。
- 手機版設計團隊區塊已改為橫向卡片展示；吉慶、桃花源、光之城等多團隊頁面可左右滑動查看。
- 單一團隊頁面（如光之瀧、大觀）不顯示左右滑動提示。
- 手機版配置圖卡片已改為橫向滑動展示，並新增「左右滑動查看配置圖」提示。
- 配置圖卡片點擊放大功能維持正常。
- 手機頁尾已新增快速導覽：
  - 回到商品系列
  - 查看工程進度
  - 回到首頁
- 手機頁尾按鈕右側已保留空間，避免被回到上方按鈕遮住。
